About StockGetter

This is the POC of the app that allows user to get all stock data in one place. It uses plain JavaScript, Bootstrap 5, Ag-Grid, Plotly, JSZip, and FileSaver to implement the app. Stock data is from fcsapi.com and www.alphavantage.co. The app utilizes a Card class to represent the card div object.
